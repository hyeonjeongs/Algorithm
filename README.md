# Algorithm

### ✨알고리즘 스터디
1. 1일1알고리즘을 목표로 매일 하나씩 풀고자 햔다.
2. 다시 풀어야하거나 좋은 문제는 주석처리한다.
3. 여러번 풀 경우, 원래 풀이 주석처리 후 다시 푼다.
4. 백준, 프로그래머스 문제 업로드한다.
